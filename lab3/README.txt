https://www.alphavantage.co/query?function=GLOBAL_QUOTE&symbol=IBM&apikey=demo       -   Валидный запрос
- Я использовал метод GET
- Запрос состоит из заголовка, в нём у меня URL и Querry-параметры
- Ответ я получил JSON в теле, и в заголовках код 200
100-е - информационные, запрос принят/обрабатывается
200-е - успешные
300-е - Перенаправления, сервер сообщает что ресурс переместился
400-е - Ошибка в запросе, сервер получил запрос, но не может обработать
500-е - Ошибка сервера, внутри него самого, запрос не может быть выполнен
